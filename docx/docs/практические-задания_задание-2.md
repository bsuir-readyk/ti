# Потоковое шифрование

Затрагиваемые темы:

- Потоковые криптосистемы.
- Генераторы ключей на основе сдвиговых регистров с линейной обратной связью.
- Примитивный многочлен.

Изучить теоретический материал [теория_задание№2.docx](#главная-страница_практические-задания_задание-2_теоретический-материал).

Выполнить задание согласно выданному [варианту](#главная-страница_практические-задания_задание-2_варианты-2).

*Для успешной защиты*:

- Корректно работающая программа, в которой учтены все требования согласно заданию.
- Отчет в печатном виде.
- Тестовые файлы.
- Знание теоретического материала по теме работы.

*Содержание отчета*:

- ФИО, номер группы, номер задания.
- Условие задачи.
- [Пример работы LFSR по тактам](#главная-страница_практические-задания_задание-2_пример-lfsr) согласно выданного варианта (тактов д.б. не менее 60ти ).
- Скрины, подтверждающие корректную работу программы.
