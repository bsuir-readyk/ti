# Простейшие шифры

Затрагиваемые темы:

- Перестановочные алгоритмы шифрования.
- Подстановочные алгоритмы шифрования.

Изучить теоретический материал [практикум_криптография.pdf (стр.4-12)](#методические-материалы), Виженер_СтолбцУлучш.docx.

*Для успешной защиты:*

- Корректно работающая программа, в которой учтены все требования согласно заданию.
- Отчет в печатном виде.
- Знание теоретического материала по теме работы.

*Содержание отчета*:

- ФИО, номер группы, номер задания.
- Условие задачи.
- [Тесты](#главная-страница_практические-задания_задание-1_тесты-1).
- Скрины, подтверждающие работу программы с файлами.
