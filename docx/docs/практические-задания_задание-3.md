# Криптосистемы с открытым ключом

Затрагиваемые темы:

- Криптосистема RSA.
- Криптосистема Рабина.
- Криптосистема Эль-Гамаля.

Изучить теоретический материал [практикум_криптография.pdf, стр.14-20](#методические-материалы).

Выполнить задание согласно выданному [варианту](#главная-страница_практические-задания_задание-3_варианты-3).

*Для успешной защиты*:

- Корректно работающая программа, в которой учтены все требования согласно заданию.
- Отчет в печатном виде.
- Тестовые файлы.
- Знание теоретического материала по теме работы.

*Содержание отчета*:

- ФИО, номер группы, номер задания.
- Условие задачи.
- Показать работу трех алгоритмов (быстрое возведение в степень, поиск первообразных корней, расширенный алгоритм Евклида) согласно [примеру](#главная-страница_практические-задания_задание-3_примеры-алгоритмов). Числа, выбранные для иллюстрации работы алгоритмов у всех студентов должны быть уникальны.
- Скрины, подтверждающие корректную работу программы.
