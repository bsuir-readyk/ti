# Практические задания

Данные практические работы являются обязательными к выполнению для допуска к экзамену.

Максимальная оценка за работу - 9 баллов.

Система оценки выполненной работы:

- Оценка максимальна, если работа выполнена и сдана в срок, установленный дедлайном.
- После выдачи каждого следующего задания максимальная оценка за предыдущее минус 2 балла т.е. если выдано задание №3, то максимальная оценка за задание №1 - (9 - 4) = 5. Если выдано задание №4, то максимальная оценка за задание №1 - 4 балла.

Очередь для сдачи практических работ будет формироваться следующим образом:

- Работа должна быть выполнена (полностью работающая программа + отчет) и отправлена накануне аудиторного занятия до 20.00. Работы, отправленные после 20.00, проверяться не будут.
- На адрес [s.boltak@bsuir.by](mailto:s.boltak@bsuir.by) необходимо отправить прямую ссылку на репозиторий GitHub с исходным кодом и отчетом. Код не должен быть в архиве. Тема письма - номер группы, номер задания. Например, 451001, лр2. В самом письме обязательно указать ФИО.
